# Connect-X-Networking-Dashboard
## 📊 Project Overview

This project focuses on analyzing ConnectX dataset to provide insights into:

* Overall performance metrics
* Category-wise analysis
* Regional trends
* Time-based performance tracking

The dashboard helps users understand key patterns and make data-driven decisions efficiently.

## 🚀 Features

* 📈 Interactive Power BI dashboard
* 📊 Key KPIs (Total Sales, Revenue, Profit, Quantity)
* 🌍 Region-wise performance analysis
* 📦 Category and segment insights
* 📅 Monthly and yearly trend analysis
* 🎯 Dynamic filters and slicers

## 🛠️ Tools & Technologies

* **Microsoft Excel**

  * Data cleaning & preprocessing
  * Data formatting
  * Preparing dataset for analysis

* **Power BI**

  * Data modeling
  * DAX calculations
  * Interactive dashboard creation

## ⚙️ How to Use

1. Clone or download the repository
2. Open the Excel file to explore the dataset
3. Open the `.pbix` file in Power BI Desktop
4. Refresh data if required
5. Use filters to interact with the dashboard
